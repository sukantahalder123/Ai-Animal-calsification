# Ai-Animal-calsification
The AI-Animal-Classification project is an image classification system built with PyTorch and Flask, designed to identify animals from uploaded images using a custom-trained AI model.

Key Features:
Trained on a custom dataset of 15 animal categories.

Built with MobileNetV2 using transfer learning.

Supports image uploads via REST API for real-time classification.

Stores and retrieves classification history from a PostgreSQL database.

Search functionality to find predictions by label.
